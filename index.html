An Algorithm for Speech Inpainting on MFCCs Using Deep Neural Networks | Listening Examples

All speakers are contained in the test set which means they have not been seen by the DNNs before. The clean audio files used to create the following audio examples were taken from the VCTK database [1].

The following methods were used to create the audio examples:

  • AudioUNet: Using the waveform-based AudioUNet trained on the lower-bandwidth data (see [2]).
  • Lower-Bandwidth $\mathrm{DNN}_\mathrm{LB}$: Using our proposed MFCC-based DNN trained on the lower-bandwidth data (see [3]).
  • Uniform $\mathrm{DNN}_\mathrm{U}$: Using our proposed MFCC-based DNN trained on the uniform data containing randomly distributed rectangular gaps (see [3]).
WN represents the case when white noise is used as excitation. CP denotes the cheated phase case. The filled gaps step was used for all audio examples. A detailed description of how the audio examples were generated can be found in [3].
